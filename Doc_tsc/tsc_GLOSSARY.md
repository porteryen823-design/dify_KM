# TSC 術語對照

> 此檔用於補強知識庫檢索，將常見中英文名稱、別名與用途整理在一起。

| 中文名稱 | 英文名稱 | 常見別名 | 說明 |
|---|---|---|---|
| 交通管制 | Traffic Control | 交管 | 控制車輛避讓、路權、繞路等規則。 |
| 路權 | Right of Way | ROW | 控制某站點或路段是否可由單一車輛通行。 |
| 群組 | Group | Group Protection | 將易互相干涉的點位設為同群組保護。 |
| 路口點 | Junction | 交管點、路口區 | 多路線匯集處的交通控制點。 |
| Keep 點 | Keep Point | Keep | 車輛可略過不停留的點。 |
| Go 點 | Go Point | Go | 車輛必須實際走到的點。 |
| Area ID | Area ID | Area | 可能跨區、跨群組、跨貨架的儲位或區域識別。 |
| `BySourePort` | `BySourePort` | Source-based dispatch | 依來源點進行區域切割調度。手冊原文如此拼寫。 |
| `ByDestPort` | `ByDestPort` | Destination-based dispatch | 依目的點進行區域切割調度。 |
| Remember Me | Remember Me | 記住帳號 | 暫存登入帳號，避免部分情境重複輸入。 |
